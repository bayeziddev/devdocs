DevDocs is an open-source static site generator designed for developers. Write in Markdown, and let the engine generate blazing-fast, mobile-ready documentation and technical blogs instantly.

Why choose DevDocs?

⚡

Insanely Fast

Built with Vanilla JavaScript and zero heavy frontend frameworks. Pages load instantly, ensuring a perfect Lighthouse score.

📝

Markdown Driven

Focus entirely on your content. Write standard Markdown files and DevDocs automatically handles the routing, parsing, and formatting.

🌓

Built-in Dark Mode

Includes an intelligent theme toggler that respects system preferences and saves user choices seamlessly via local storage.

🔍

SEO Optimized

Generates clean, semantic HTML with automated canonical URLs, Open Graph tags, and structured JSON-LD data for maximum visibility.

How it works

1. Drop your Markdown files into the /content folder.

2. Run the build script via Node.js.

3. Deploy the static HTML files anywhere (GitHub Pages, Vercel, Netlify).
